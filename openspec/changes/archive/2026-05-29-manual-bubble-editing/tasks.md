## 6. Commit pipeline

- [x] 6.1 Add `commitEditSession() async` that follows the 12-step sequence in `design.md` §D10. Use the existing OCR service and `TranslationService.translate(...)` (per-page path). Implement the **final-state no-op short-circuit (Step 3)**: when the final working set has the same bubble IDs, order, and `boundingBox` values as `originalSnapshot`, skip OCR, translator, and cache writes, restore `originalPageState`, and clear `editSession`. Implement the **empty-set short-circuit (Step 4)**: when the final working set is empty, skip OCR and translator entirely and write `.translated([])`. The OCR axis and the translate-input axis are otherwise independent: **every non-deleted bubble is included in the translate call**; only the OCR axis has a "skip" branch (skipped bubbles reuse their snapshot `text`). OCR classification at commit MUST be derived from **current `boundingBox` vs `originalSnapshot` `boundingBox`** (exact `CGRect.equalTo`, no tolerance) — NOT from `dirtyBubbleIds` (which is a UI cache only). Unit-test with a fake `TranslationService` and a fake OCR service covering: (a) commit with no effective final-state change → OCR/translator/cache are never called and session closes; (b) commit with only new boxes → OCR runs only on new, translate input = all non-deleted; (c) commit with only deletions (some surviving) → OCR not called, translate input = surviving bubbles (assert the deleted ids are NOT in the call); (d) commit with only reorder → OCR not called, translate input = all bubbles in the **new** order, argument order matches the post-reorder `index` values; (e) move-then-undo-to-original → no-op branch, OCR and translator are not called, and transient `isManual` changes are not committed; (f) thrown error during translate → page state `.error`, `editSession` remains non-nil with original snapshot intact; **(g) commit with all bubbles deleted (surviving count = 0) → OCR never called, `TranslationService.translate(...)` never called, `page.state` becomes `.translated([])`, `editSession` cleared, and this succeeds even when the translator fake is configured to throw on any call (proving the short-circuit avoids the translator entirely).**
